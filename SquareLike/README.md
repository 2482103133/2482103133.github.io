# SquareLike
一个简单的俄罗斯方块练手项目,在原有的俄罗斯方块的基础上,添加了打飞机,炸弹,修复块等新(gui)奇(chu)功能.<br/>奇怪的东西不断掉下来???正义的铁拳打的你措手不及!!  
## 游戏操作:
↑:方块变形  
↓:加速下降  
←→:方块左右移动  
空格:功能键  
## 方块类别:
### 炸弹
可以根据吨量炸掉黑色方块  
功能键:快速下落到底部
### 修复块  
生成一个3X3方块  
功能键:快速下落到底部
### 普通方块  
到最低点将被固定成黑色块  
功能键:快速下落到底部
### 轰炸机
碰到障碍物将爆炸,效果等同于炸弹  
功能键:投放一个1x1的小炸弹  
### 召唤块
轰炸召唤召唤一排1x1小方块  
### 蛇蛇:
吃掉所有食物化身金龙(yu)  
### 炮塔Turret:
有点蠢的自动机器人  
### 小坦克:
迎接炮弹的洗礼吧    
## 下载与使用:
Download as ZIP 然后解压,在浏览器中打开Main.html.
